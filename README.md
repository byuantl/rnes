# rnes
I wanted to work on a fun project and I thought it would be really cool to be able to play NES games using my own emulator! It was also a way to get more familiar with using Rust and work toward becoming a rustacean :crab:!

Currently only the 6502 CPU has been implemented, the PPU, APU, and Cartridge are still a work in progress.
